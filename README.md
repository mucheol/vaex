# 🌐 VAEX

**VAEX 회사 메인 홈페이지**  
VAEX 디지털 자산 거래소를 소개하는 **기업 메인 웹사이트**로,  
거래소 개요, 서비스 설명, 비전, 로드맵, 매뉴얼 안내 등 정보를 제공하는 정적 웹 페이지입니다.  
전체 퍼블리싱과 주요 화면 기획 의도를 반영하여, 사용자가 VAEX 거래소를 한눈에 이해할 수 있도록 구성했습니다.

---

### 📘 프로젝트 개요
- **목적:** VAEX 거래소의 브랜드 아이덴티티, 서비스 소개, 로드맵, 이용 매뉴얼 등을 한 페이지/멀티 섹션 구조로 전달  
- **특징:** 메인 랜딩 페이지를 중심으로 한 기업형 홈페이지 구조, 거래소의 비전과 서비스 정보를 시각적으로 표현
- **작업기간:** 2022.02 ~ 2022.04  
- **작업참여도:** 퍼블리싱 100%  

---

### 🧱 제작방식
- **Frontend:** HTML + CSS + JavaScript  
- **Plugin:** Fullpage.js (섹션 단위 스크롤 연출)
- **Size:** Desktop + Laptop + Mobile (반응형 지원)  

---

### 🛠 주요 구현 내용

#### 🧭 레이아웃 및 페이지 흐름
- Fullpage.js를 활용한 섹션 기반 풀스크린 스크롤 인터페이스 구현  
- 메인 히어로 섹션에서 VAEX의 핵심 메시지(플랫폼 소개, 비전)를 강조하는 구조
- 아래로 스크롤하면서 순차적으로 **서비스 소개 → 로드맵 → 기능/장점 → 매뉴얼 안내**를 확인할 수 있도록 플로우 구성  

#### 🎨 퍼블리싱 & 인터랙션
- 시맨틱 태그(`header`, `section`, `footer`)를 활용하여 기업 사이트 구조를 명확하게 구분  
- CSS 애니메이션 및 트랜지션을 활용해 스크롤 시 텍스트/이미지 등장 효과 구현  
- 고정 내비게이션(또는 인디케이터)을 통해 현재 섹션 위치를 시각적으로 표시  
- JavaScript와 Fullpage.js 옵션 설정으로 스크롤 속도, 이징(easing), 네비게이션 동작을 커스터마이징  

#### 📑 콘텐츠 구성
- VAEX 거래소 소개: 플랫폼 개념, 글로벌 디지털 자산 거래소로서의 포지셔닝 설명 
- 비전 및 로드맵: 서비스 확장 계획 및 향후 방향성을 타임라인 형식으로 표현  
- 주요 기능: 거래, 자산 관리, 보안, 지원 등의 핵심 기능을 카드 단위 UI로 정리  
- 매뉴얼/가이드 링크 섹션: 이용자가 상세 매뉴얼 페이지로 이동할 수 있도록 안내  

---

### ⚙️ 기술적 포인트
- **Fullpage.js 커스터마이징:**  
  - 섹션 전환 애니메이션, 네비게이션, 앵커 링크 설정 등 옵션 최적화
  - 섹션별 배경 이미지/컬러를 다르게 적용하여 시각적 구분감 강화  
- **반응형 퍼블리싱:**  
  - Desktop, Laptop, Mobile 해상도에 따라 타이포그래피와 레이아웃이 자연스럽게 재배치  
  - 모바일 환경에서는 풀스크린 섹션 비율과 최소 터치 영역 확보에 중점  
- **브랜딩 및 일관성:**  
  - VAEX 브랜드 컬러와 톤을 유지하면서, 텍스트와 아이콘을 조합한 정보 전달 중심 UI 설계 [web:18][web:19]  
  - 단일 페이지 내에서 일관된 컴포넌트 스타일(버튼, 카드, 타이틀)을 적용하여 통일감 있는 화면 구현
