How do I deposit the coin?

1. Choose Balance - Account at the top right of the screen
  • Balance can only be checked with login status.
2. Select the coin deposit you want to send.
3. Deposit
  • You can receive the corresponding coin by delivering the QR code or address name to the other party.
  • Please check the coin name and address when delivering it.
  • The address on the screen above is the ID address for test. Please check it again when you send it.
